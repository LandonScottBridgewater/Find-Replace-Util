# Find-Replace-Util

this script scans all descendants and replaces the query with the specified value.

## Example

```
git clone https://github.com/LandonScottBridgewater/Find-Replace-Util/
cd Find-Replace-Util
python3 find_replace.py
```
```
Enter target substring: import spyder
Enter replacement substring: import spyder_e
Enter path: ~/Documents/spyder
```
